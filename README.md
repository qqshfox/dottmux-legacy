Hanfei Shen's dottmux files
==========================

Installation
------------

```sh
git clone https://github.com/qqshfox/dottmux.git ~/.tmux
cd ~/.tmux
make install
tmux
```

Dependencies
------------

* tmux 1.6
* iTunes on Mac OS X Lion (Optional)
